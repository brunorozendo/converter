# A simple converter ISO-8859-1 to UTF-8 ### Usage Require `Java 1.8` ```bash java -jar converter.java ``` ### Download [converter.jar](https://github.com/brunorozendo/converter/releases/download/1.2/converter.jar) . The input file (txt) must have a list of files with the full path. Ex (Linux): ``` /path/of/file/File.java /path/of/file/File1.html /path/of/file/File2.xml /path/of/file/File3.txt /path/of/file/File4.c ``` Ex (Windows): ``` C:\Users\\Deskop\files\File.java C:\Users\\Deskop\files\File1.html C:\Users\\Deskop\files\File2.xml C:\Users\\Deskop\files\File3.txt C:\Users\\Deskop\files\File4.c ```
